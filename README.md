# Crypto Price Dashboard

A simple beginner-friendly dashboard that shows live cryptocurrency prices using the CoinGecko API.

Built with HTML, CSS, JavaScript.

## Features

- Live crypto prices
- Auto-refresh every 10 seconds
- Price-change color indicators (green = up, red = down)
- Clean card UI

### Tech Stack

- HTML
- CSS
- JavaScript
- API: CoinGecko (no API key required)
